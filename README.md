# Interactive Drum Kit 🥁

A dynamic, fully functional web-based drum machine that responds to both mouse clicks and keyboard presses.

## Description
This application allows users to play different drum sounds by interacting with the on-screen buttons or using specific keys on their keyboard. It showcases advanced JavaScript event handling, including attaching multiple event listeners, detecting keydown events, triggering local audio files, and applying momentary CSS animations for visual feedback.

## Features
- Interactive UI with visual click/press feedback
- Keyboard and mouse event listeners
- Integrated audio playback via JavaScript

## Technologies Used
- **Frontend:** HTML5, CSS3
- **Logic:** JavaScript (Advanced DOM manipulation)

## How to Run Locally
1. Clone this repository.
2. Ensure the `sounds` folder (containing the audio files) is intact in your directory.
3. Open `index.html` in your web browser.
4. Click the buttons or press the corresponding keys on your keyboard to play the sounds.
